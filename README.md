# Points
Ejemplo de carga de varios puntos desde un Geojson


## AVISO PRIVACIDAD:
Este repositorio es público. Por favor, no introduzcas nombres completos, datos personales reales ni información sensible o que no te sientas cómodo/a compartiendo con el resto del grupo.

Respecto a la localización, las coordenadas no deben corresponder a una ubicación exacta ni a un domicilio real. Puedes utilizar una localización aproximada o ficticia, por ejemplo, el centro de tu ciudad, municipio o provincia.
